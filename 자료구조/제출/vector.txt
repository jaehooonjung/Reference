vector
C++에서 지원하는 자료구조 자료형

1.필요 헤더파일

#include <vector>

2.변수 생성 방법

std::vector<int> ivec;

3.정보 저장 방법.

ivec.push_back(1);

for(int i = 0 ; i < 10 ; i++)
{
	ivec.push_back(i);
}

4.정보 탐색 방법
for(auto iter = ivec.begin(); iter != ivec.end(); iter++)
{
	cout << *iter << endl;
}

4.정보 제거 방법

ivec.pop_back();

ivec.clear();

for(auto iter = ivec.begin(); iter != ivec.end(); iter++)
{
	if (*iter == 4)
	{
		iter = ivec.erase(iter);
	}
}


5.복사 생성자
std::vector<int> vec1;
for(int i = 0 ; i < 10 ; i++)
	vec1.push_back(i);
std::vector<int> vec2(vec1);
for(int i = 0 ; i < 10 ; i++)
{
	cout<< vec1[i] <<endl;
	cout<< vec2[i] <<endl;
	cout << "===============" << endl;
}

그외 기능들


-ivec.size() = 원소의 갯수 반환
-ivec.empty() = 원소가 비어있는지 확인(비어있으면 true를 반환)
-ivec.at(i) = vector의 i번째 원소를 참조함 ( 0번째 부터 시작 )
-ivec[i] = 배열처럼 사용 가능
-ivec.back() = vector의 마지막 원소를 참조

-vector1 == vector2 = 두 백터 변수를 비교하여 모두 같으면 true 하나라도 다를경우 false를 반환
-vector1 != vector2 = 두 백터 변수를 비교하여 하나라도 다를경우 true 모두 같으면 false를 반환
